EndNote设置导出参考文献格式为中文国标GBT7714

发表于 2021-02-20 分类于 科研 阅读次数： Valine：

笔者使用的版本是最新的EndNote 20，其他版本大同小异。

依次打开：Tools -> Output Styles -> Open Style Manager。其他版本可能有差异，只要找到Open Style Manager即可。

在列表里找到，勾上即可。

而现在还没有安装，找不到Chinese Standard，所以要下载。点击Get More on the Web...。

打开后搜索关键词Chinese，下载Chinese Standard GBT7714，这是目前国内中文期刊、高校毕业论文比较通用的国标格式。

鉴于外网访问可能比较慢，不方便的朋友可以直接使用笔者的网盘资源：

链接：https://pan.baidu.com/s/1L-eOwrdh_YPfTNBx6uEosA
提取码：qc8s

双击下载好的.ens后缀文件，用默认的EndNote样式编辑器打开，选择File->Save As...。

点击保存即可：

然后就可以选择导出格式了。快乐~ (*^▽^*)

这时候打开Style Manager，也可以发现Chinese Standard已经被勾上了：

祝大家磕盐愉快~
